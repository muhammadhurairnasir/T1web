# Lab Task 3 - Express.js with EJS

Convert HTML/CSS layout into Express.js application with EJS templates.

## Requirements Completed

✅ Convert HTML/CSS to Express.js  
✅ Set up clean routes  
✅ Create Express project  
✅ Configure EJS template system  
✅ Break pages into views and partials  

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Pages

- `/` - Homepage
- `/checkout` - Checkout page
- `/order-success` - Order confirmation

## Structure

```
lab-task-3/
├── routes/          - URL routes
├── views/           - EJS templates
│   └── partials/    - Reusable components
├── public/          - CSS, images, JS
└── server.js        - Express server
```
